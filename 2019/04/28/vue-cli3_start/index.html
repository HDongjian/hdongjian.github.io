是否使用Class风格装饰器？
即原本是：home = new Vue()创建vue实例

使用Babel与TypeScript一起用于自动检测的填充? yes

路由使用历史模式? 这种模式充分利用 history.pushState API 来完成 URL 跳转而无须重新加载页面

使用什么css预编译器？ 看个人习惯

tslint: typescript格式验证工具
eslint w…: 只进行报错提醒；
eslint + A…: 不严谨模式；
eslint + S…: 正常模式；
eslint + P…: 严格模式；

建议选择正常模式
